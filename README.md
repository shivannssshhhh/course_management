# course_management
Course Management System
This project is a Spring Boot based Course Management System developed to perform CRUD operations on course data.
The application uses MySQL as the backend database and follows a layered architecture using Controller, Service, and Repository.

Technologies Used
Java

Spring Boot

Spring Data JPA

MySQL

Hibernate

Maven

Functionalities
Add course details

View all courses

Update course details

Delete course details

Database Used
MySQL

sql
Copy code
CREATE DATABASE course_management;
How to Run
Start MySQL server

Configure database in application.properties

Run the Spring Boot application

Access APIs at http://localhost:8081

Conclusion
This project demonstrates the use of Spring Boot REST APIs with MySQL database integration for managing course information.
